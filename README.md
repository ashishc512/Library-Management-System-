A Library Management System developed using Java, JDBC and MySQL.

## Features
- Register new members
- Prevent duplicate member IDs
- View available books from database
- Issue books
- Return books
- Fine calculation for overdue returns

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Database Tables
- Members
- Books
- Records
